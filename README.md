## Setup
### Dataset setup
- Download the dataset from [Hugging Face](https://huggingface.co/datasets/Wenyan0110/Multimodal-Dataset-Image_Text_Table_TimeSeries-for-Financial-Time-Series-Forecasting).
- Save the dataset in the `data/` folder at the project root directory.
- Run `cookbooks/EDA/news_eda_1.ipynb` to generate
  `data/multi-modal-dataset/stock_scores_news_1.csv`.
- If you store the dataset in a different location, update the paths in `src/config.yaml` accordingly.
